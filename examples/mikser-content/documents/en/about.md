Acme Co is a fictional agency we use throughout the mikser
documentation. It builds content-rich client sites where every
page is owned by editorial, every URL is durable across decades,
and every interaction is composed from independent layers.

This about page is a `meta.layout: page` document — the simplest
shape. The Vue apps render it via `PageView.vue` (the fallback
layout). mikser's own rendering uses the universal
`layouts/page.html.hbs` template.
